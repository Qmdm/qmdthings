# 编写你自己的函数

R 语言允许用户创建 function 模式的对象。事实上，R 系统自带的大多数函数（如 mean(), var()）本身也是用 R 语言编写的

**定义格式**：`name <- function(arg_1, arg_2, ...) expression`

**返回值**：函数体（expression）通常是一个用大括号 {} 括起来的分组表达式，其最后一个计算出的表达式的值即为该函数的返回值

```r
> twosam <- function(y1, y2) {
  n1 <- length(y1); n2 <- length(y2)
  yb1 <- mean(y1); yb2 <- mean(y2)
  s1 <- var(y1); s2 <- var(y2)
  s <- ((n1-1)*s1 + (n2-1)*s2)/(n1+n2-2)
  tst <- (yb1 - yb2)/sqrt(s*(1/n1 + 1/n2))
  tst
}
> tstat <- twosam(data$male, data$female); tstat
```

## 定义新的二元运算符

你可以定义形如 `%anything%` 的二元运算符

定义时需要给函数名加上引号，例如 `"%!%" <- function(X, y) { ... }。`定义后即可像 `X %!% y` 这样使用

## 命名参数和默认值

**调用方式**：调用函数时，参数可以按位置传递，也可以使用 `name=object` 的形式。如果指明了名称，参数的顺序可以任意排列

**默认值**：可以在定义时为参数指定默认值。如果调用时省略了这些参数，则使用默认值。默认值可以是任意表达式，甚至可以涉及该函数的其他参数

## "..."参数

**参数传递**：... 是一个特殊的参数，允许一个函数将其接收到的参数直接传递给另一个函数

**应用场景**：这在绘图函数中非常常见，例如将图形参数从自定义函数传递给底层的 `par()` 函数

**访问方法**：可以使用 `list(...)` 将这些额外参数收集到一个命名列表中

## 删除打印数组中的所有名称

```r
> temp <- X
> dimnames(temp) <- list(rep("", nrow(X)), rep("", ncol(X)))
> temp; rm(temp)
```

使用如下所示的函数 `no.dimnames()` 作为“包装函数”可以更方便地实现相同的结果。这也说明了某些高效实用的用户函数可以非常简洁。

```r
no.dimnames <- function(a) {
## Remove all dimension names from an array for compact printing.
d <- list()
l <- 0
for(i in dim(a)) {
d[[l <- l + 1]] <- rep("", i)
}
dimnames(a) <- d
a
}
> no.dimnames(X)
```

## 函数内的赋值

**局部性**：函数内部进行的普通赋值（`<-`）是局部且临时的，在退出函数后这些变量就会消失，不会影响调用环境中的同名对象。

**全局赋值**：如果需要进行永久或全局赋值，可以使用 `<<-` 运算符或 assign() 函数

## 作用域

**变量分类**：函数体内的符号分为三类：**形式参数（formal parameters）**、**局部变量（local variables）**和**自由变量（free variables）**。
自由变量如果被赋值，就会变成局部变量.

**词法作用域 (Lexical Scope)** ：R 采用词法作用域，这意味着自由变量的值是由函数被创建时所属的环境来决定的。

**闭包与状态**：利用词法作用域和 `<<-`运算符，可以创建具有“可变状态”的函数（例如模拟一个银行账户，在多次调用间保持余额信息）。

```r
open.account <- function(total) {
  # 返回一个包含三个函数的列表，这些函数共享当前的 total 变量
  list(
    # 存款函数
    deposit = function(amount) {
      if(amount <= 0)
        stop("存款金额必须为正数！\n")
      total <<- total + amount  # 使用 <<- 修改外部环境中的 total
      cat(amount, "已存入。当前余额为", total, "\n\n")
    },
    # 取款函数
    withdraw = function(amount) {
      if(amount > total)
        stop("余额不足！\n")
      total <<- total - amount  # 使用 <<- 修改外部环境中的 total
      cat(amount, "已取出。当前余额为", total, "\n\n")
    },
    # 查询余额函数
    balance = function() {
      cat("您的当前余额为", total, "\n\n")
    }
  )
}
```

## 定制环境

**初始化文件**：站点初始化文件的位置取决于 `R_PROFILE` 环境变量的值, 通过 `Rprofile.site`（站点级）和 `.Rprofile`（个人级）文件可以定制 R 的启动环境。如果设置了环境变量 R_PROFILE_USER，则使用该变量指向的文件，而不是 .Rprofile 文件。

因此，文件的执行顺序是：`Rprofile.site`、用户配置文件、`.RData`，最后是`.First()`。后面文件中的定义会覆盖前面文件中的定义。

**特殊函数**：

- `.First()`：在会话开始时自动运行，常用于设置提示符、加载常用包等。
- `.Last()`：在会话结束时自动运行，常用于关闭图形设备或清理工作。

```r
> .First <- function() {
  options(prompt="$ ", continue="+\t") # $ is the prompt
  options(digits=5, length=999) # custom numbers and printout
  x11() # for graphics
  par(pch = "+") # plotting character
  source(file.path(Sys.getenv("HOME"), "R", "mystuff.R"))
  # my personal functions
  library(MASS) # attach a package
}

> .Last <- function() {
graphics.off() # a small safety measure.
cat(paste(date(),"\nAdios\n")) # Is it time for lunch?
}
```

## 泛型函数和面向对象

**泛型函数 (Generic Functions)** ：**S3 分发机制** 根据参数的 class（类） 来决定执行具体的操作。例如 `plot()` 和 `summary()` 对数据框和因子会有不同的反应。

你可以使用 `methods()` 函数来查看一个泛型函数支持哪些类，或者一个类支持哪些泛型函数。

**分发机制** ：通过 `UseMethod("function_name")` 实现。例如 `coef.aov` 是 `coef` 函数针对 `aov` 类对象的方法
