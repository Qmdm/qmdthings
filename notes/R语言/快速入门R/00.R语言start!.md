# R

R 可以看作是 S 语言的一种实现，S 语言是由贝尔实验室的 Rick Becker、John Chambers 和 Allan Wilks 开发的，也是 S-Plus 系统的基础。

R是一套集成软件工具，用于数据操作、计算和图形显示。除此之外，它还有

- 一个有效的数据处理和存储设施，
- 一套用于数组（尤其是矩阵）计算的运算符，
- 一个庞大、连贯、集成的数据分析中间工具集合，
- 用于数据分析显示的图形设施，可以直接在计算机上使用或在打印介质上使用，以及
- 一种开发完善、简单且高效的编程语言（称为‘S’），它包含条件语句、循环、用户自定
  义递归函数以及输入和输出功能。（实际上，系统提供的许多函数本身也是用S语言编写的。

R大约提供了25个软件包（称为“标准”和“推荐”软件包），还有更多通过
CRAN系列的互联网站点（通过https://CRAN.R‑project.org）和其他地方提供。

S（以及因此的R）与其他主要统计系统之间在哲学上存在一个重要区别。在S中，统计
分析通常是一系列步骤完成的，中间结果会存储在对象中。因此，尽管SAS和SPSS会从回归
或判别分析中给出大量输出，R会给出最小输出并将结果存储在一个fit对象中，供后续␁R␁函
数进一步查询。
