# 分组,循环,条件执行

## 分组表达式

语言特性：R 是一种表达式语言，这意味着它的唯一命令类型是返回结果的函数或表达式。 甚至赋值操作也是一个表达式，其结果是所赋的值。

组合方式：可以使用大括号 {...} 将一系列命令组合在一起。

返回值：整个分组表达式的值是该组中最后一个被计算的表达式的结果。 由于分组本身也是一个表达式，它可以被放入圆括号中，作为更大型表达式的一部分。

```r
> {1+1;2+2}
[1] 4
```

## 控制语句

### 条件执行

基本结构：标准形式为 `if (expr_1) expr_2 else expr_3`。 其中 `expr_1` 必须计算出一个单一的逻辑值（TRUE 或 FALSE）。

短路运算符：在 if 的条件判断中经常使用 `&&` 和 `||`。 与作用于整个向量的 `&` 和 `|` 不同，这两个运算符仅作用于长度为一的向量逐元素的进行计算，且仅在必要时才计算第二个参数（即“短路”特性）。

向量化版本：对于需要对整个向量进行条件判断的情况，应使用 `ifelse(condition, a, b)` 函数，它会返回一个与 condition 长度相同的向量。

### 重复执行

**for 循环**：格式为 `for (name in expr_1) expr_2`。 name 是循环变量，`expr_1` 通常是一个序列（如 1:20），`expr_2` 会随着变量在序列中的取值而重复执行。

- 提示：R 中通常使用 split() 函数将数据分组后再进行循环处理（例如分组绘图）。
- 警告：**在 R 中，for 循环的使用频率远低于编译型语言**。通常采用**“全对象” (whole object) 视角**编写的代码（如向量化操作）在 R 中运行更快、更清晰。

```r
> xc <- split(x, ind)
> yc <- split(y, ind)
> for (i in 1:length(yc)) {
plot(xc[[i]], yc[[i]])
abline(lsfit(xc[[i]], yc[[i]]))
}
```

**while 循环**：格式为 `while (condition) expr`，只要条件为真就重复执行。
**repeat 循环**：格式为 `repeat expr`，它会无条件重复执行，直到遇到跳出指令。

**break 语句**：用于立即终止任何循环。 这是退出 repeat 循环的唯一方式。
**next 语句**：用于中断当前这一次循环周期，直接跳到下一次迭代
