# R中的11个统计模型

## 统计模型的定义与公式

R 采用一种统一的符号体系（公式）来描述统计模型，其模板通常是线性模型：y=Xβ+e。

其中 y 为响应向量，X 为模型矩阵或设计矩阵，其列为 x₁, x₂, ..., xₙ，代表决定变量。通常情况下，xₙ 列的元素均为 1，表示截距项。

**公式操作符 ~**：用于定义响应变量和预测变量之间的关系，例如 response ~ predictor_terms。

**常用符号含义**：

- `+` 和 `-`：包含或排除某个项（1 代表截距项，默认包含）。
- `:`：变量间的交互作用（张量积）。
- `*`：代表全因子模型（如 A\*B 等同于 A + B + A:B）。
- `/`：代表嵌套关系（如 A/B 等同于 A + B %in% A）。
- `^n`：包含所有直到 n 阶的交互作用。
- `I()`：恒等函数，用于在公式中按算术含义使用运算符（如 I(x2) 代表 x 的平方）。

## 对比

对于分类变量（因子），R 需要将其转换为模型矩阵中的列。

- **无序因子**：默认生成 k−1 列，将后续各水平与第一水平进行对比（Treatment contrasts）。
- **有序因子**：默认生成 k−1 个正交多项式列。
- **全局设置**：可以通过 options(contrasts = ...) 修改默认的对比方式

## 线性模型

lm() 是拟合普通多元线性回归模型的核心函数。

    调用格式：fitted.model <- lm(formula, data = data.frame)。
    数据参数：data 参数非常重要，它指定了模型所需变量的首选来源地（通常是一个数据框）。

## 用于提取模型信息的通用函数

R 拟合模型后生成的对象包含大量信息，通常使用特定的“提取函数”来获取细节，而不是直接打印对象：

    summary()：打印详细的回归分析报告（包括系数、标准误、t 值、p 值和 R2）。
    anova()：通过方差分析表比较两个或多个模型。
    coef()：提取模型系数。
    residuals()：提取残差。
    predict()：对新数据进行预测。
    plot()：生成诊断图（如残差图、正态概率图等）

## 方差分析和模型比较

aov() 用于方差分析，其用法与 lm() 类似，但特别支持多误差层级的模型（如分裂区试验）。

    多层级误差：使用 Error(strata) 来指定误差项，例如 aov(yield ~ v + Error(farms/blocks))。
