# 从文件读取数据

R的输入设施需求较为严格且相对固定，通常建议用户在导入前使用其他工具（如编辑器）预处理数据以符合 R 的格式要求

## read.table() 函数

这是读取完整 **数据框(Data Frame)** 的主要函数

文件格式要求：外部文件通常需要特定格式。第一行应包含每个变量的名称；后续每一行的第一个条目为行标签，后面跟着各变量的值

默认情况下，数值项被读入为数值变量，非数值变量被读入为字符变量

如果文件中没有显式的行标签，可以使用 header=TRUE 选项，此时 R 会将第一行视为标题，并使用默认的行标签

```r
>HousePrise <- read.table("houses.data",header=TRUE)
```

## scan() 函数

这是一个比 `read.table()` 更底层的输入函数，适用于更灵活的数据读取需求

可以并行读取多个等长的向量。通常通过提供一个**虚拟列表 (dummy list)** 参数来指定要读取的各个向量的模式（如字符型或数值型）

假设数据向量长度相等且需要并行读取。进一步假设有三个向量，第一个向量为字符型，其余两个向量为数值型，文件为 input.dat。第一步是使用 scan() 函数将这三个向量读取为一个列表，如下所示。

```r
> inp<-scan("input.dat",list("",0,0))
> label <- inp[[1]]; x <- inp[[2]]; y <- inp[[3]]
> label <- inp$id; x <- inp$x; y <- inp$y
> X <- matrix(scan("light.dat", 0), ncol=5, byrow=TRUE)
```

第二个参数是一个虚拟列表结构，用于确定要读取的三个向量的模式。结果存储在 `inp` 中，它是一个列表，其元素是读取的三个向量。要将数据项拆分成三个单独的向量，请使用类似这样的赋值语句。

读取的结果通常是一个列表，用户可以通过赋值操作或挂载（attach）该列表来访问具体的变量向量

如果第二个参数是单个值而非列表，则会将整个文件读入为一个特定模式的单一向量。

## 访问内置数据集

R 自带了约 100 个数据集（位于 datasets 包中），可以直接按名称访问

- **列出数据集**: 使用 `data()` 命令可以查看当前可用的数据集列表
- **加载数据集**: 虽然大多数数据集可直接使用，但有些包仍沿用 data(数据名称) 的方式来显式加载

```r
data("Orange")
> ls()
[1] "Orange"
```

## 从其他R 包加载数据

若要访问特定包中的数据，可以使用 `package` 参数，例如 `data(Puromycin, package="datasets")`

```r
> data(Puromycin, package="datasets")
> ls()
[1] "Puromycin"
> head(Puromycin)
  conc rate   state
1 0.02   76 treated
2 0.02   47 treated
3 0.06   97 treated
4 0.06  107 treated
5 0.11  123 treated
6 0.11  139 treated
```

如果一个包已经通过 `library()` 函数被挂载到搜索路径中，其包含的数据集会自动变为可用状态

## 编辑数据

R 提供了简单的交互式工具来对已读取的数据进行微调。

- **edit()函数**: 对数据框或矩阵调用此函数会弹出一个类似电子表格的界面。完成编辑后，需要将结果重新赋值给对象（如 `xnew <- edit(xold)` ）

- **fix()函数**：这是一个快捷方式，相当于执行 `xold <- edit(xold)`，直接修改原始数据集

- **输入新数据**：也可以通过 `edit(data.frame())` 打开空白表格来手动录入新数据

实际上有一些第三方包支持更加强大的数据清洗(如tidyverse qwq)
